## DQN jan-ken-pon

### What is this ?

Tiny DQN sample codes for chainer, TensorFlow and Keras inspired by [Alog Lab](http://blog.algolab.jp/post/2016/08/01/tf-dqn-simple-1/)'s simple code.

This code learns how to win Rock-paper-scissors games.


### Usage

Training

```
$ python train.py
```

Test

```
$ python test.py
```
